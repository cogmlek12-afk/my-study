\# 클래스 기초 개념 정리



\## 1. 클래스와 객체의 정의



\- 클래스(Class): 객체를 만들기 위한 설계도

\- 객체(Object): 클래스를 통해 실제로 만들어진 실체



예:

\- 클래스 = 화장품 설계도

\- 객체 = 립스틱, 크림 같은 실제 제품



\---



\## 2. 클래스 구성 요소



\### 1) 필드 (Field)

객체의 상태를 나타내는 데이터

예: 이름, 브랜드, 가격



\### 2) 메서드 (Method)

객체의 기능

예: 정보 출력 기능



\### 3) 생성자 (Constructor)

객체 생성 시 초기값 설정



\---



\## 정리



클래스는 설계도이고,

객체는 그 설계도로 만든 실제 결과물이다.
